# myblog


### install [node](https://nodejs.org) or use [nvm](https://github.com/creationix/nvm)

	brew install npm

	brew install nvm

### install [hexo](http://hexo.io) 

	npm install hexo-cli -g

### clone code

	git clone git@github.com:xiaoxie/myblog.git

### create hexo blog

	hexo init blog

### merge code

	cd blog

	mv -R myblog/* .

### install package

	npm install

### run hexo

	hexo server
