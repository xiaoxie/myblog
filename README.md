# myblog


### install node

	brew install npm

	brew install nvm

### install hexo 

	npm install hexo-cli -g

### clone code

	git clone git@github.com:xiaoxie/myblog.git

### create hexo blog

	hexo init blog

### merge code

	cd blog

	mv -R myblog/* .

### install package

	npm install

### run hexo

	hexo server
